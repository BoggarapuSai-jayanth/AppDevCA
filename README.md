# AppDevCA - CGPA Calculator App

A simple Android application built with Kotlin that calculates CGPA (Cumulative Grade Point Average) from three subject grades.

## 📱 Description

AppDevCA is a mobile application designed to help students quickly calculate their CGPA by inputting grades from three subjects. The app performs a simple average calculation and displays the result to the user.

## ✨ Features

- **Easy-to-use Interface**: Clean and intuitive UI built with Android XML layouts
- **Multi-subject Support**: Calculate CGPA from three subjects
- **Input Validation**: Ensures all subject grades are entered before calculation
- **Real-time Results**: Displays calculated CGPA with formatted output
- **User-friendly Feedback**: Toast notifications for user guidance

## 🛠️ Technologies Used

- **Language**: Kotlin
- **Platform**: Android
- **UI Framework**: Android LinearLayout
- **Build System**: Android Gradle
- **Minimum SDK**: Android (target varies)

## 📋 Requirements

- Android Studio (latest version recommended)
- Android SDK
- Kotlin support enabled
- Minimum API level for target Android version

## 🚀 Getting Started

### Prerequisites

1. Install Android Studio
2. Clone or download this repository
3. Open the project in Android Studio

### Installation

1. Open the project in Android Studio
2. Sync Gradle files
3. Connect an Android device or start an emulator
4. Run the application

### Usage

1. Launch the app on your Android device
2. Enter the CGPA values for Subject 1, Subject 2, and Subject 3
3. Tap the "Calculate CGPA" button
4. The app will display your calculated CGPA as a toast notification

## 📊 How It Works

The app calculates CGPA using the formula:
```
CGPA = ((Subject1/10) + (Subject2/10) + (Subject3/10)) / 3
```
